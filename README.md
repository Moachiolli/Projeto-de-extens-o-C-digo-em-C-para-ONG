# Projeto-de-extens-o-C-digo-em-C-para-ONG
Um modelo de código completo em C, estruturado e pronto para o seu trabalho. Ele foi projetado para ser simples, funcional e fácil de explicar na sua apresentação ou relatório, utilizando exatamente os conceitos de lógica de programação vistos em sala de aula (como switch-case, funções, structs, vetores e laços de repetição).
